# BERSn UI (Phase 1)

Phase 1 UI is intentionally minimal and demo-focused.

## Pages
- `/login`
- `/dashboard`
- `/calculation`

## Run

```bash
npm run dev
```

Default API base URL is `http://localhost:8080`.

Override API URL if needed:

```bash
VITE_API_URL=http://localhost:8080 npm run dev
```
